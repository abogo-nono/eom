What is EOM?

Eye On Metadata (E.O.M) is an open-source desktop application that displays, exports and removes metadata from a media (images, videos, audio, pdf, ...). EOM is cross-platform so it can work on Windows, Linux, and MacOS as well.

How do you extract/display metadata from a media?

Follow these steps to display metadata from one or many file

  1. Click on extract
  2. Select the extraction type
  3. Select the file or the directory
  4. Click on the extract button

How do you remove metadata from one or many files?

To remove one metadata from media, you must follow these steps

  1. Click on remove
  2. Select the remove type
  3. Select the file or the directory
  4. Click on the remove button

How do you export metadata of file(s) to your local storage?

You can export metadata of one or many files into a file stored in your local disk

  1. Click on extract
  2. Select the extract type
  3. Select a file or directory by clicking the browse button
  4. Click on export
  5. Select the path and the name of the file which will contain the data
  6. Click on save

How do you report a bug, contribute, or suggest features?

  1. Open this link EOM repo
